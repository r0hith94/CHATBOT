# AI Chatbot Web Application 🤖

A fully functional AI chatbot web application that allows users to chat with an AI powered by Groq's Llama 3.1 model.

---

## Overview

This project demonstrates a full-stack AI chatbot where users can interact in real-time with an intelligent AI. The backend is powered by Django, and the frontend provides a smooth and interactive chat interface.

---

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript (interactive chat UI)  
- **Backend:** Django (Python web framework)  
- **API:** Groq Llama 3.1 model for AI responses  
- **Database:** SQLite (Django default)  
- **Version Control:** Git & GitHub  

---

## Features

- ✅ Real-time chat interface  
- ✅ AI-powered responses using Groq Llama 3.1  
- ✅ Modern gradient UI with animations  
- ✅ CORS enabled for API access  
- ✅ REST API backend using Django  
- ✅ Fully functional for production use  

---

## How It Works

1. User types a message in the chat box  
2. Frontend sends the message to Django backend via API  
3. Backend calls Groq API to get AI response  
4. Response is sent back to frontend  
5. Message is displayed in the chat UI
